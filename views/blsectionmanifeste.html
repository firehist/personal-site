Manifeste d'un Front-End

Le plus important principe est

de placer, loin devant, les besoins

UTILISATEURS

par rapport aux besoins développeurs.

Evolution constante

et JavaScript non-bloquant

Sans JavaScript ou CSS, une application web se doit

de fonctionner au détriment de sa beauté!

La performance

est très importante et n'ai pas souvant reflétée

par les environnements de travail des développeurs.

Un développeur Front-End doit apprendre la base:

Le JavaScript

avant l'abstraction avec jQuery, AngularJS, YUI, ...
